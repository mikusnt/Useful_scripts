command - en - pl
pwd - print working directory - aktualny katalog roboczy
ls - list directory contexts - zawartość danego katalogu
cd - rename actual directory - zmiana aktualnego katalogu
rmdir - remove empty directories - usuwa puste katalogi
mkdir - make directories - tworzy foldery
rm - remove files and directories - usuwa pliki i foldery
cp - copy files and directories - kopiowanie plików i folderów
mv - move and rename files - zmiana położenia, nazwy plików
qrep - print lines matching a pattern - wypisuje linie w pliku pasujące do wzorca
find - search for files in a directory hierarchy - wyszukiwarka plików w określonej strukturze katalogów
cat - concatenate and print files - łączy i wypisuje zawartość plików
shutdown - halt, power down, reboot machine - resetuje, wyłącza, uruchamia ponownie maszynę
file - determine file type - typ pliku
which - localization of command - lokalizacja polecenia
du - file space usage - zajetosc na dysku poszczegolnych plikow
uniq - report or omin repeated lines - zgłasza lub omija powtórzone linie
wc - print newline, word, byte counts for file - podaje liczbę wierszy, słów, bajtów danego pliku
head - output first part of file - zwraca poczatek pliku o okreslonej dlugosci
tail - output last part of file - zwraca koniec pliku o okreslonej dlugosci
pr - splits the data into pages - podzial tekstu na strony
tr - translate character - zamiana znakow na inne
less - file perusal filter for crt viewing - podzial tekstu na fragmenty wyswietlane na monitorze
printenv - print all or part of environment variable - wyswietla fragment lub wszystkie zmienne srodowiskowe
calendar - reminder service
cal - calendar of actual month
